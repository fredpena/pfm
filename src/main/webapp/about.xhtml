RUP Management System es una aplicación web para la gestión de proyectos RUP construida como Proyecto Fin de Máster del Máster en Ingeniería Web (MIW) de la Universidad Politécnica de Madrid. Para su implementación se ha seguido la metodología ligera Scrum mediante 4 Sprints de 2 semanas cada uno entre abril y junio de 2014
